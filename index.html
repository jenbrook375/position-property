static postions as per html flow. these elements are considered to be not positioned
relative position is the same as static unless you give it extra values
giving values to top, right, bottom and left will move the element. these values are relative to it's default position
behaves like relative except that it is relative to its parent, top, right, bottom and left give it a new position
fixed position will move to a new position and stay there even if content around it moves or is being scrolled
